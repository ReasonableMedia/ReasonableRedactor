The full ReasonableRedactor tool runs locally on your Windows computer and permanently deletes personal data from the PDF file itself, not just visually. It also includes:

  • Phone number redaction (UK and international formats)
  • Name redaction with a custom list you control
  • Bcc header removal
  • Allowlisting so you can protect some addresses but not others
  • Clean or blackout redaction styles

It requires Python to be installed once. After that it is one double-click to run, and opens in your browser just like this tool.
